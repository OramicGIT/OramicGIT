OramiX System: Linux для Windows-пользователей

OramiX System — это минималистичная современная ОС на ядре Linux с дизайном в стиле Fluent и Rectify11, запускается как LiveCD в Windows или VM. Не DOS, а лёгкая, красивая Linux-экосистема для автоматизации, обучения и everyday-задач.

  • CLI- и GUI: Удобство командной строки и аккуратной графической оболочки.
  • Автоматизация: Bash-скрипты, интеграция с утилитами OramiX, расширяемость.
  • Совместимость: Быстрый запуск на ПК и в виртуалках.
  • Обучение и эксперименты: Для изучения Linux, автоматизации и безопасных тестов.
  • LiveCD: Не требует установки, всегда быстро и удобно.

O-Setup-SRV: Генератор Батч-Установщиков

O-Setup-SRV — инструмент для создания .bat-установщиков Windows для быстрой автоматизации развёртывания ПО.

  • Создание .bat-файлов: Быстрое создание скриптов для установки программ.
  • Массовое развёртывание: Локально и через интернет.
  • Безопасность: Проверка подписи скриптов.
  • Веб-интерфейс: Управление библиотекой установщиков.
@echo off
call o-setup-srv /install Chrome Office7 AdobeReader

FlimClient: Дискорд-клиент для ботов (Fun & Jokes)

FlimClient — это не чат-бот, а клиент для входа по Discord bot token и общения как бот — для фана, шуток и экспериментов. Позволяет смотреть, как выглядит чат от лица бота, отправлять сообщения и тестировать реакции.

  • Вход как бот: Используйте Discord token и общайтесь как бот.
  • Fun and Jokes: Не предназначен для реального управления сервером, только для развлечения и экспериментов!
  • Простой интерфейс: Вдохновлён Fluent и Rectify11 — плавные анимации, скругления, “стеклянные” элементы.

PhunLite: Минималистичный 2D/3D-движок

PhunLite — лёгкий движок для быстрого прототипирования игр и визуализаций, в стиле Fluent и с простым API.

  • Быстрое прототипирование: Простые игры за минуты.
  • Минимум ресурсов: Легко работает даже на старых ПК.
  • Кросс-совместимость: Windows и OramiX System.
  • Готовые шаблоны: Учитесь на примерах.
