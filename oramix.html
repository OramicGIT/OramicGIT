В разработке

OramicCOS: Переосмысление Контроля

OramicCOS — это манифест возврата к истокам эффективного взаимодействия с компьютером. Архитектура на основе batch-скриптов, встроенный CLI, быстрая загрузка и работа без лишней графики. Для энтузиастов, администраторов, всех, кто ценит минимум ресурсов, максимум возможностей.

  • CLI-контроль: Управляйте файлами, процессами, настройками через мощные команды.
  • Автоматизация: Скрипты для рутинных задач, развертывания, обслуживания.
  • Гибкость: Добавляйте свои утилиты, расширяйте функционал.
  • Образование: Идеально для изучения основ ОС и командной строки.
  • LiveCD: Запуск без установки — быстро, безопасно, удобно.

Система развивается, появляются новые команды, расширяется поддержка оборудования. Присоединяйтесь к сообществу OramiX, делитесь скриптами, изучайте примеры и создавайте свои инструменты!

O-Setup-SRV: Сервер и Генератор Батч-Установщиков

O-Setup-SRV — революционный инструмент для создания и хранения кастомных установщиков Windows в формате .bat! Просто выберите нужные программы, опции и получите готовый скрипт. Запускайте установку на любом ПК или распространяйте через корпоративную сеть.

  • Генерация .bat-файлов для установки любого ПО;
  • Групповое развертывание через сервер (локально или онлайн);
  • Безопасность: цифровые подписи, контроль целостности;
  • Веб-интерфейс для управления и истории установок.

Пример использования:
@echo off
call o-setup-srv /install Chrome Office7

FlimClient: Ваш AI-Компаньон и Бот для Windows

FlimClient — универсальный чат-бот и автоматизированный помощник. Общайтесь, управляйте задачами, получайте справку по OramiX и Windows, интегрируйте FlimClient в свои скрипты или проекты. Поддержка персонализации, команд, сценариев, диалогов и даже интеграция с сетевыми сервисами.

  • Интерактивный чат прямо на рабочем столе OramiX (или в web-интерфейсе);
  • Поддержка команд для управления ПК, поиска, автоматизации;
  • Расширяемые навыки — пишите свои скрипты и команды для бота;
  • Интеграция с O-Setup-SRV, PhunLite и другими проектами OramiX.

Пример: FlimClient: "Создай батч-установщик для Notepad++ и VLC!"

PhunLite: Минималистичный 2D/3D-движок

PhunLite (или Phun) — лёгкий движок для прототипирования игр, визуализации и экспериментов. Минимум зависимостей, быстрый старт, открытый исходный код. Идеально для обучения, курсов, хакатонов и творчества на старых или слабых ПК.

  • Быстрое создание простых игр и визуальных приложений;
  • Минимум графики и медиа — всё можно добавить позже;
  • Совместимость с Windows и OramicCOS;
  • Примеры: "Hello, World", платформер, визуализация данных.
